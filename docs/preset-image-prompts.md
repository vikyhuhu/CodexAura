# 内置预设生图规范

这些提示词用于生成 CodexAura 的 16:9 壁纸。共同构图要求：画布 2560×1440；只使用一个连续、干净的场景，禁止相框、卡片、拼贴、胶带、贴纸和画中画；左侧约 65%～70% 是低细节、低对比、低饱和的文字安全区，只允许柔和渐变、薄雾或极淡远景；主视觉缩小并集中在最右侧约 20%～28%；避免纯黑、大面积高饱和色、锐利高光、密集纹理和多个视觉焦点；不得包含可读文字、字母、数字、软件界面、标志或水印。

## 葫芦娃

以参考图中的经典中国二维动画为角色和画风依据：辨识度明确的七个葫芦娃、手绘造型、墨线轮廓和水粉山岭背景。七兄弟作为一个紧凑团队站在画面右侧，整体缩小并上移，人物脚部必须高于画面 78% 的位置，使底部输入区不遮挡身体；红娃略靠前，其余六位紧密排在后方，使用平静自信的站姿，不使用战斗特效。蛇精与蝎子精位于相对的左侧远景山台并面向七兄弟：蛇精严格参考蓝灰肤色、锐利长脸、黑色几何短发与三环发饰、小型蓝冠、高金色领口和蓝绿色纹样服装；蝎子精严格参考红橙色长脸、黑色胡须、红色头饰、深色身体及带金纹的蓝色高领造型。反派整体较小并稍微降低对比度，中央约 50% 保持无人、低细节的正文安全区。背景维持暖象牙色晨雾、极淡灰绿色远山和右上角克制的葫芦藤，整体低饱和、低反差、安静怀旧。

## 发财程序员

原创复古动画海报风，奶油黄、朱红、玉绿色调，丝网印刷颗粒和轻微旧纸质感。右侧是一位原创的快乐小程序员吉祥物，坐在复古电脑旁，周围是抽象金币、盆栽和几何好运符号；不使用任何真实软件界面、品牌或汉字。左侧保持温暖浅色、低对比和稀疏装饰，适合覆盖深色文字。活泼、俏皮、清晰轮廓，无文字、标志、水印。

## 奥特曼

经典银红奥特曼主题，英雄轮廓、面部、胸前计时器和服装纹样具有明确辨识度。只画一位奥特曼，以克制的守护姿势站在最右侧，比例不要过大；背景是暖象牙色到浅蓝灰色的连续晨曦天空，底部右侧仅保留极淡城市剪影，英雄背后可有一条几乎融入天空的柔和光弧。不出现怪兽、战斗、爆炸、星空、控制台、雷达、徽章或重复人物。采用简化的手绘水粉与很轻的复古印刷颗粒，低饱和、低反差、安静可靠。

## 元气食堂

原创可爱食物动画风，奶油白、番茄红、南瓜橙和叶绿色，柔和手绘描边与轻微纸张颗粒。右侧是原创的小厨师吉祥物和几种拟人化家常食物伙伴，在温暖的小食堂窗口前挥手；食物造型通用，不模仿现有角色。左侧是浅奶油色墙面和极淡瓷砖纹理，不放盘子、角色或醒目装饰。轻松、治愈、适合长时间使用，无文字、标志、水印。

## 二次构图修正

如果首轮生成的角色过大，使用同一张图编辑：保持原有角色、风格、配色、光照和宽高比不变；将全部角色及高对比道具整体缩小约 25%～35%，移动到画面最右侧，使其只占右侧约三分之一；自然补绘空出的背景，并让左侧约 58% 成为连续、低细节、低对比的安全区；不要新增文字、标志、水印、UI 或角色。

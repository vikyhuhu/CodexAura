# 内置预设生图规范

这些提示词用于生成 CodexAura 的 16:9 原创壁纸。共同构图要求：画布 2560×1440；左侧约 58% 保持低细节、低对比、无主体，供 Codex 导航和正文使用；角色及视觉焦点集中在右侧 30%～35%；边缘可延展裁切；不得包含文字、字母、数字、标志、水印或现有 IP 角色。

## 剪纸山海守夜人

原创东方民俗剪纸插画。深朱红、炭黑与少量暖金配色，层叠纸张纤维和手工镂刻边缘。右侧是一位原创的圆润小守夜人，戴兽纹兜帽、提暖光灯笼，身后有抽象云纹、山形和温和的瑞兽剪影；气氛安静、可靠、略带童趣。左侧使用大面积暗红纸纹和极淡云纹，不放角色或高对比元素。精致但不写实，无文字、标志、水印。

## 发财程序员

原创复古动画海报风，奶油黄、朱红、玉绿色调，丝网印刷颗粒和轻微旧纸质感。右侧是一位原创的快乐小程序员吉祥物，坐在复古电脑旁，周围是抽象金币、盆栽和几何好运符号；不使用任何真实软件界面、品牌或汉字。左侧保持温暖浅色、低对比和稀疏装饰，适合覆盖深色文字。活泼、俏皮、清晰轮廓，无文字、标志、水印。

## 星光小队长

原创复古科幻动画插画，靛蓝、深紫、青蓝和暖橙星光。右侧是一位原创的小小星际队长，穿简洁的圆形太空服、拿小型星图投影器，身后是抽象星球环、彗星和玩具般飞船；角色造型不对应任何现有作品。左侧是大面积平静深空渐变，仅保留微弱星尘，避免亮星和主体。电影感、温暖冒险气氛，无文字、标志、水印。

## 千禧信使

原创 2000 年代初期网络幻想风，柔和天蓝、银灰、薰衣草紫和少量青柠色，半透明塑料、像素光点与气泡质感。右侧是一位原创的圆润信使精灵，背小邮包，身边漂浮抽象信封、星星和果冻状装置；不出现真实聊天软件、品牌图标或可读 UI。左侧保持明亮干净的天空渐变和极淡网格，低细节、低对比。怀旧又清爽，无文字、标志、水印。

## 元气食堂

原创可爱食物动画风，奶油白、番茄红、南瓜橙和叶绿色，柔和手绘描边与轻微纸张颗粒。右侧是原创的小厨师吉祥物和几种拟人化家常食物伙伴，在温暖的小食堂窗口前挥手；食物造型通用，不模仿现有角色。左侧是浅奶油色墙面和极淡瓷砖纹理，不放盘子、角色或醒目装饰。轻松、治愈、适合长时间使用，无文字、标志、水印。

## 二次构图修正

如果首轮生成的角色过大，使用同一张图编辑：保持原有角色、风格、配色、光照和宽高比不变；将全部角色及高对比道具整体缩小约 25%～35%，移动到画面最右侧，使其只占右侧约三分之一；自然补绘空出的背景，并让左侧约 58% 成为连续、低细节、低对比的安全区；不要新增文字、标志、水印、UI 或角色。
